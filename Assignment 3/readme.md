# Raretifact

## Overview

Raretifact is a simple web application developed using **HTML, CSS, and JavaScript**. The website allows users to explore and sell old or antique artifacts through an easy-to-use interface.

The project was created as part of the **Full Stack Development Lab (FSDL)** practical assignments to demonstrate frontend web development concepts.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Features

- Responsive user interface
- Home page showcasing antique artifacts
- Browse available artifacts
- Sell old artifacts
- Simple navigation
- Interactive UI using JavaScript
